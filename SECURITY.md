# Security Policy

## Reporting a Vulnerability
If you discover a security issue, please do not open a public issue.

- Email: `community@hexology.me`
- Include: repro steps, impact, and any relevant logs/screenshots

We will acknowledge receipt and work on a fix as quickly as possible.

## Secret Handling
- Do not commit private keys, certificates, or `.env` files.
- If a secret is accidentally committed, rotate it immediately and remove it from git history if required.
