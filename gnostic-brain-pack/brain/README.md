# brain/

## Directory roles

### core/
Slow-changing identity and thesis.

### state/
Fast-changing operational truth.

### reflection/
Meta-learning, drift, and recurring patterns.

### daily/
One file per day. Tactical state compression.

### decisions/
ADR-style records for decisions that matter.

## Rule of thumb
- core = months
- state = days
- reflection = weeks
- daily = today
- decisions = until superseded
