# PRIORITIES.md

## Current priorities
1. Finish the organizer IA cut: worlds are now the obvious home, so route names and remaining legacy surfaces need to catch up.
2. Push world ownership deeper into tournaments, lobbies, and matches until the IA is honest.
3. Remove brand and product drift between BOARD intent and Hexology-era interior surfaces.

## Why these are the priorities
- Product truth depends on real organizer primitives, not better copy.
- The new world slice only matters if users keep that context after creation and stop getting pulled back into legacy language.
- Brand incoherence is now a direct trust problem, not a cosmetic issue.

## Explicit deprioritized work
- New wallet or World-native side quests that do not strengthen the organizer wedge.
- Premium/player vanity expansion.
- Broadening game count or mod complexity before the chess-first host flow is strong.

## Killswitch
If one of the above priorities is wrong now, what should replace it?
- If organizer adoption is not the real wedge, stop using world/server language and return the product narrative to what the code actually is.
